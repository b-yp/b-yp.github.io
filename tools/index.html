            # 浏览器插件

            没有插件的浏览器是没有灵魂的 浏览器推荐：Chrome > 新版edge > 火狐 > 其他 有一个心愿浏览器 (opens new window)，听说很不错，可以试试

            • Tampermonkey

              毫无疑问，宇宙最强浏览器插件。这是一个脚本管理器，后边会介绍一些脚本

            • Infinity 新标签页（pro）

              这是一款可以让你的新打开的标签页更美观的插件，话不多说，上张图吧

            • 简悦

              简悦 - 为你提供「如杂志般沉浸式阅读体验」的扩展，这是官网对它的介绍，它的功能远不止增强阅读体验这么简单

            • Google 翻译

              这个好像Chrome内置了，不过我平时还是喜欢用下面这个

            • 侧边翻译

              划词翻译用得很舒服

            • Adblock Plus

              一款广告拦截器

            • 图片助手(ImageAssistant)

              可以一键提取网页上所有的图片，进行批量下载

            • 滴答清单 - Todo

              待办和任务管理工具

            • cVim

              脱离鼠标操作浏览器，上手有些难度（用过vim的除外）

            • Dark Reader

              黑色主题，夜猫子必备

            • Evernote Web Clipper

              如果你使用印象笔记的话，用它剪藏会很方便

            • Stylus

              可以调整网页外观的用户样式管理器，比如更改网站主题，安装皮肤

            • 二箱

              多引擎以图搜图

            • FireShot

              捕捉网页截图，不过我一般都用 Snipaste

            • 琉神转

              老司机工具箱

            • 破解右键锁

              顾名思义

            • 谷歌访问助手

              你品

            • 类似的网站

              我用它上P站，然后发现了很多*****网站

            • 二维码生成器

              将当前网页，或者是网页上的文本、链接，或者是你输入的任何内容转换成二维码

            • 书签侧边栏

              将书签以侧边栏的形式展示在页面左侧，没什么卵用

            • Isometric Contributions

              将github的提交以立体的形式展示出来

            • Octotree

              树形展示github仓库目录结构
